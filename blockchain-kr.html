VALICO B.C

VALICO B.C는 블록체인 기본 개념에 충실하며, 개발 용이성이 보장되는 자체 블록체인 프로토콜, ‘VIA CHAIN’을 개발하고 있습니다.

Why VIA CHAIN?

  • 비즈니스에 적합한 블록체인 시스템

    비즈니스에서 요구되는 다양한 자료 형태와 연산을 위한 프토토콜 설계가 가능합니다.

  • 빠른 블록의 완성에 초점

    완성된 블록을 빨리 사용할 수 있게 함으로써, 블록에 대한 연산 속도를 개선시켰습니다.

    ※ VIA CHAIN Test specification : 4core 4GB(cloud VM), 3 Node
  • 표준 기술 사용해 시스템 확장성 극대화

    VIA CHAIN의 Node가 Java로 만들어졌다 하더라도 API를 C++, C#, python 등 다양한 언어로 개발이 가능합니다.

  • Block 데이터 간의 Link 연결 구조

    외부 API의 요청에 대해서 빠른 검증 및 조회 내역을 제공할 수 있습니다.

Application Examples

  • Via Chain Feature for Embedded
    • VTTP를 통해 검증 및 합의 과정을 간단하고 빠르게 처리
    • JAVA만으로 구성돼 시스템 구동 시 VM외에는 불필요
    • 블록체인 저장공간을 File I/O 환경으로 구축
    • Linux 및 RTOS에서 지원하는 네트워크 프로토콜(gRPC)을 활용
    • 개별 Node를 외부와 분리 상태 운영, 사생활 정보 특수 상황 제외 외부 유출 차단
  • VIA CHAIN Feature for IoT
    • 기술 표준에 해당하는 XML을 Data Format으로 활용
    • 블록이 가진 고유 Link를 통해 해당 블록의 유효성 검증 신속 처리
    • 블록의 주요 데이터는 요구 사항에 따라 암호화 처리 가능
